# 📊 Sales Dashboard Analysis (Excel)

## 🔍 Project Overview
This project analyzes sales performance using an interactive Excel dashboard.

## 📁 Dataset
- Superstore Sales Dataset
- Includes orders, customers, regions, and product categories

## 📊 KPIs
- Total Sales
- Total Orders
- Total Quantity Sold
- Average Order Value

## 📈 Key Insights
- Revenue is concentrated among top-performing products
- Certain regions contribute more significantly to overall sales
- Customer segments show different purchasing behaviors

## 🛠 Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Dashboard Design)
